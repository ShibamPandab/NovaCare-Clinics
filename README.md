## Overview

**NovaCare Clinics** is a polished, multi-section **healthcare website** built to showcase how a modern medical clinic can present its services, specialists, and patient-facing tools online. The platform combines clean clinical branding with practical UI patterns — department directories, doctor profiles, an interactive appointment booking form, and an emergency contact system — to create a complete **patient experience** from first visit to confirmed consultation.

Designed around a calm blue, teal, and navy palette with crisp typography, NovaCare Clinics serves as a portfolio-grade example of **modern medical website** design that balances trust, clarity, and functionality.

---

## Features

- 🚑 **24/7 Emergency Contact Bar** — persistent critical trauma hotline and clinic hours at the top of the page
- 📊 **Live Statistics Counters** — patients treated, certified doctors, departments, and satisfaction score
- 🩺 **Department Directory** — six medical specialities (Cardiology, Neurology, Orthopedics, Pediatrics, Oncology, General Medicine) with descriptions and stats
- 👩‍⚕️ **Doctor Profiles with Search & Filter** — searchable specialist grid filterable by department, showing ratings, reviews, and live availability
- 📅 **Interactive Appointment Booking Form** — full validation for name, email, phone, date, department, doctor, and time slot, with a telehealth consultation option
- ✅ **Booking Confirmation Modal** — generates a reference ID and summarizes appointment details on submission
- 🧭 **Why Choose Us Section** — accreditation badges and clinical trust pillars
- ❓ **FAQ Accordion** — expandable answers covering booking, insurance, emergencies, and pricing
- 💬 **Patient Testimonials** — star-rated reviews from verified patients
- 🗺️ **Stylized Location & Contact Section** — clinic address, hours, and an illustrated map widget
- 📩 **Newsletter Signup** — footer subscription form for health updates
- 🟢 **Floating WhatsApp Support Widget** — expandable chat-style contact panel

---

## Design Highlights

- **Clinical Color System** — clean whites paired with clinic blue, navy, and teal accents for a trustworthy medical tone
- **Editorial Display Typography** — bold display headings paired with a readable sans-serif body font
- **Motion-Enhanced Interactions** — smooth accordion, modal, and mobile-menu animations powered by Framer Motion
- **Card-Based Information Architecture** — departments, doctors, and trust pillars presented as scannable, hover-responsive cards
- **Status & Trust Indicators** — live availability tags, accreditation badges, and rating displays throughout

## Performance & Responsiveness

Built on **Vite** for fast development and optimized production builds, NovaCare Clinics is fully **responsive across desktop, tablet, and mobile**. The layout adapts from multi-column department and doctor grids on larger screens to a streamlined single-column flow on mobile, with a collapsible navigation drawer ensuring the booking flow and emergency contact remain accessible on any device.

## Installation

```bash
# Clone the repository
git clone https://github.com/ShibamPandab/NovaCare-Clinics.git

# Move into the project directory
cd NovaCare-Clinics

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173` by default.
